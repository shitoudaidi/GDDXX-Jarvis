# Iteration 12 - Cockpit visual hierarchy

1. [Aesthetic] Conversation content sat too close to the rail edge; the history gutter is now 16px.
2. [Aesthetic] Message text was visually weak at 13px; it now uses 14px type with a 1.62 reading rhythm.
3. [Conversation] Speaker labels and timestamps were too small to scan; message metadata now has a 10px floor.
4. [Aesthetic] Channel and timestamp text disappeared into the background; secondary contrast is now stronger.
5. [Design] The empty history state floated in a full-height column; it is now a bounded, centered surface.
6. [Design] The open composer had no visible editing boundary; it now has a restrained field border and fill.
7. [Function] Keyboard focus inside the composer was hard to locate; focus now adds a cyan edge marker.
8. [Design] Keyboard, replay, and music controls competed with the main voice action; they now use a quieter tone.
9. [Aesthetic] The main voice action lacked sufficient hierarchy; its ring and restrained glow are now stronger.
10. [Aesthetic] News titles were cramped and faint; rows now use 12px type, 1.5 line height, and more vertical space.
